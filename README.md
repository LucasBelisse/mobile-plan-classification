Classificação de Planos de Celular: Megaline

Este projeto faz parte do meu portfólio de Ciência de Dados e foca no desenvolvimento de modelos de aprendizado de máquina para prever qual plano de telefonia (Smart ou Ultra) é mais adequado para um cliente, baseando-se em seu histórico de uso.

## 📝 Contexto do Problema
A operadora Megaline deseja analisar o comportamento de seus clientes e recomendar um dos seus dois planos mais recentes. O objetivo é automatizar esse processo com alta precisão para garantir a satisfação do cliente e a eficiência da empresa.

## 📊 Resultados e Performance
Foram testados três modelos principais (Árvore de Decisão, Floresta Aleatória e Regressão Logística) com ajuste de hiperparâmetros.

- **Melhor Modelo:** Random Forest (Floresta Aleatória)
- **Acurácia Alcançada:** **80.2%** ✅
- **Métrica Alvo (Threshold):** 75%



## 🛠️ Tecnologias e Técnicas
- **Linguagem:** Python
- **Bibliotecas:** Pandas, Scikit-Learn
- **Técnicas:**
  - Divisão de dados (Treino, Validação e Teste)
  - Ajuste de hiperparâmetros (Tuning)
  - Avaliação de modelos de classificação

## 📁 Estrutura do Projeto
- `mobile_plan_recommendation.ipynb`: Notebook com toda a análise e desenvolvimento do modelo.
- `users_behavior.csv`: Base de dados contendo chamadas, minutos, mensagens e tráfego de internet.
